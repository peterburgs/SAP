CREATE A NEW PROJECT

Once you log in successfully, you will be navigated to the Project list activity which contains all your projects.
1. On Project list activity, tap the icon “+” on toolbar (top of the app).
2. on Create project activity, fill out project name and key, then tap “Create project” button.
Figure 10 Create new project
3. You will be navigated to the Project list activity. The new project will appear soon.
