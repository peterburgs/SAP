CREATE A NEW SPRINT

On Project list activity, choose a project.
1. On the project board, tap the floating action button (FAB) on the right-bottom corner.
          Figure 11 Open floating action button menu
 
2. Choose “Sprint.”
3. On Sprint list activity, tap the icon “+” on the toolbar (top of the app).
Figure 12 Sprint list activity
 
4. On Create sprint activity, enter the sprint name (required) and goal (optional).
Figure 13 Create sprint
 
5. Tap “Create sprint” button.
