SIGN UP AN ACCOUNT

On the Welcome activity, choose “Sign Up”.
Figure 2 Sign up an account
 
1. You must fill out all required fields, in which:
·        Username: the name you want to display.
·        Email: your email, we will send verification code via this email.
·        Password: must include lower case, upper case, number & special characters.
 
Below is an example.
Figure 3 Filled signup form
 
2. Tap “Sign Up” button to proceed.
3. Open your mailbox, the verification code will be sent immediately.
Figure 4 Verification code
 
4. Fill the code and tap “Confirm” button.
Figure 5 Enter verification code and confirm
 
 5. You will be navigated to the Login activity now.
